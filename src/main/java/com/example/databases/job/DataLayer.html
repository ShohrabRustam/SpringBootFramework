Data Layer

    Basically we have there layer in Spring Boot.
  1. Presentation Layer: Presentation layer present data and the application features to the user. This is the layer where in all the controller classes exist.
  2. Service Layer : Service layer is where business logic resides in the aplication. Task such as evaluations, decison making, processing of data is done at this layer.
  3. Data Access Layer :Data Access layer is the layer where all repository class reside
