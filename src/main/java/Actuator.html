What is Actuator in SpringBoot?

Actuator is an integral part of spring boot applications. It provides built-in production-ready features to monitor and manage your application to minitod and manage our application.

Important: Its give us the ability to monitor and manage our application

Features:

  • Built in end point
  • Ability to view real time matrics. end point to see th matric: /matric
  • Customizable

Some end-point for Actuator

  • /health : useful to checking the status of the application , such as database connectivity, disk space and custom health check.
  • /info : Display application version and git commit information.
  • /metrics : show matrics which allowed us to understand the performance and behaviour of our running application
  • /logger : Allow us to query and modify the logging lever in our application.
  • /beans : Provides a complete list of the spring beans in our application.
  • /shutdown : Allow our application to be gracefully shut down.
