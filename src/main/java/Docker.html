Docker

Docker is and open source platform that allow us to automate the development, scaling and management the application using containerization.

Vertual machines

  1. VMs act like separate computer in your computer.
  2. Each Vertual machine behave like a separate computer.
  3. VMs are created and managed by virtualization software.
  4. It provides a flexible and scalable way to utilize hardware resource.

Difference between Vertual machines and docker

  • Docker is lightweight and resource-efficient.
  • Docker is almost instant startup as no OS boot required
  • Docker utilized fewer system resource
  • Docker is independent of host OS.
